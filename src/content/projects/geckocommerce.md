description: |
  GeckoCommerce es una plataforma de comercio electrónico desarrollada con Laravel, diseñada para ofrecer una solución completa tanto para la administración de una tienda virtual como para la experiencia de compra de los clientes.

  El proyecto incorpora un panel administrativo moderno con gestión integral del catálogo, usuarios, pedidos, pagos, blog, contenido promocional y configuraciones generales. Además, implementa buenas prácticas de desarrollo, una arquitectura escalable y herramientas orientadas a mejorar la seguridad, mantenibilidad y experiencia de usuario.

content: |

  ## Características Principales

  - Plataforma de comercio electrónico completa desarrollada con Laravel.
  - Catálogo jerárquico mediante familias, categorías, productos y variantes.
  - Gestión de múltiples imágenes por producto con reordenamiento.
  - Variantes de productos mediante opciones y características (color, talla, etc.).
  - Carrito de compras y proceso completo de checkout.
  - Gestión de pedidos, pagos y transacciones.
  - Integración preparada para múltiples pasarelas de pago.
  - Sistema de envíos con administración de direcciones del cliente.
  - Panel administrativo moderno y responsive.
  - Gestión de usuarios, clientes, roles y permisos.
  - Perfil de usuario tanto para clientes como administradores.
  - Blog integrado con flujo de revisión y publicación.
  - Gestión de portadas promocionales para la tienda.
  - Notificaciones internas en tiempo real.
  - Exportación de información a Excel, CSV y PDF.
  - Auditoría automática de acciones realizadas en el sistema.
  - Soporte para modo claro y oscuro.
  - Diseño adaptable para dispositivos móviles.

  ## Aspectos Técnicos Destacados

  - Desarrollo con Laravel 12 y PHP 8.2.
  - Arquitectura MVC basada en buenas prácticas.
  - Componentes dinámicos utilizando Livewire 3.
  - Interfaces modernas con Tailwind CSS y Flowbite.
  - Autenticación mediante Laravel Jetstream con soporte para doble factor (2FA).
  - Sistema de roles y permisos implementado con Spatie Permission.
  - Auditoría automática de operaciones y registros de acceso.
  - Gestión de archivos e imágenes con soporte para múltiples cargas.
  - Slugs automáticos para URLs amigables.
  - Validación de formularios tanto en cliente como servidor.
  - Exportación profesional mediante Laravel Excel y Laravel PDF.
  - Organización modular y código reutilizable para facilitar el mantenimiento y futuras ampliaciones.
