El proyecto fue desarrollado utilizando PHP, MySQL, HTML, CSS y JavaScript, implementando una estructura funcional enfocada en ofrecer una experiencia de compra sencilla y atractiva para los usuarios.

  El sistema permite administrar productos organizados por categorías, mostrar información detallada de prendas, gestionar imágenes y facilitar la interacción del cliente mediante un flujo de selección de productos y pedidos.

  Además, incorpora funcionalidades como autenticación de usuarios, gestión de sesiones, conexión segura mediante PDO, validación de formularios, organización dinámica del catálogo y un proceso de solicitud de pedido con opciones de entrega o recojo en tienda.

  Este proyecto demuestra conocimientos en desarrollo web con PHP, manejo de bases de datos relacionales, interacción cliente-servidor y construcción de soluciones digitales orientadas a pequeños negocios comerciales.

content: |
  ## Descripción General

  Constantine es una plataforma web diseñada para una tienda de prendas de vestir, permitiendo mostrar productos disponibles mediante un catálogo organizado y facilitar la gestión de pedidos de los clientes.

  El sistema fue desarrollado como una solución personalizada para digitalizar la presentación de productos de una tienda física, permitiendo mejorar la experiencia del usuario mediante una interfaz visual y un flujo de navegación intuitivo.

  ## Características Principales

  - Catálogo dinámico de prendas de vestir.
  - Visualización detallada de productos.
  - Organización de productos mediante categorías.
  - Gestión de imágenes asociadas a productos.
  - Sistema de usuarios con manejo de sesiones.
  - Conexión a base de datos utilizando PDO.
  - Consultas preparadas para mayor seguridad.
  - Validación de formularios.
  - Gestión de pedidos realizados por clientes.
  - Selección de modalidad de entrega:
    - Delivery con dirección registrada o nueva dirección.
    - Recojo en tienda mediante selección de local.
  - Interfaz adaptable para dispositivos móviles.
  - Diseño enfocado en experiencia de usuario.

  ## Módulos Implementados

  ### Catálogo de Productos

  Permite mostrar las prendas disponibles con información relevante como nombre, descripción, precio, imágenes y categoría, facilitando la exploración del catálogo por parte del usuario.

  ### Gestión de Categorías

  Implementa una organización de productos mediante categorías, permitiendo clasificar las prendas para mejorar la navegación dentro de la tienda virtual.

  ### Gestión de Usuarios

  Incluye funcionalidades relacionadas con usuarios y sesiones, permitiendo mantener información del cliente durante la interacción con la plataforma.

  ### Gestión de Pedidos

  Permite al usuario seleccionar productos y completar una solicitud de pedido indicando la modalidad de entrega preferida.

  El sistema contempla dos alternativas:

  - Delivery:
    Permite utilizar la dirección registrada del cliente o ingresar una nueva ubicación.

  - Recojo en tienda:
    Permite seleccionar uno de los locales disponibles para retirar el pedido.

  ## Aspectos Técnicos Destacados

  - Desarrollo backend utilizando PHP nativo.
  - Manejo de base de datos MySQL.
  - Uso de PDO para conexión segura con la base de datos.
  - Organización de archivos PHP sin framework.
  - Gestión de sesiones para usuarios.
  - Uso de consultas preparadas para prevenir inyección SQL.
  - Implementación de diseño responsive.
  - Integración de componentes visuales mediante JavaScript.
  - Separación de estilos CSS y scripts JavaScript para mejorar mantenimiento.

  ## Objetivos del Proyecto

  - Crear una solución digital para una tienda de ropa.
  - Aplicar conocimientos de desarrollo web con PHP y MySQL.
  - Diseñar una experiencia de usuario sencilla para la compra de productos.
  - Implementar un catálogo funcional adaptable a diferentes dispositivos.
  - Desarrollar una aplicación web cercana a un entorno comercial real.
