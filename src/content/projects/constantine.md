title: Constantine

short_description: Tienda virtual de prendas de vestir desarrollada en PHP con catálogo dinámico, gestión de productos y flujo de pedidos.

description: |
  Constantine es una aplicación web desarrollada para la presentación y gestión de un catálogo virtual de prendas de vestir. El sistema permite mostrar productos organizados por categorías, administrar información de prendas y ofrecer a los clientes una experiencia sencilla para seleccionar productos y realizar solicitudes de pedido.

  El proyecto fue desarrollado utilizando PHP, MySQL, HTML, CSS y JavaScript, implementando una estructura organizada sin framework backend, con enfoque en seguridad, mantenimiento del código y experiencia de usuario.

content: |

  ## Características Principales

  - Catálogo dinámico de prendas de vestir.
  - Visualización detallada de productos.
  - Organización de productos mediante categorías.
  - Gestión de imágenes asociadas a productos.
  - Sistema de usuarios con manejo de sesiones.
  - Selección de productos para realizar pedidos.
  - Configuración de modalidad de entrega:
  - Delivery con dirección registrada o nueva dirección.
  - Recojo en tienda mediante selección de local.
  - Validación de formularios.
  - Interfaz adaptable para dispositivos móviles.
  - Diseño enfocado en experiencia de usuario.

  ## Implementación Técnica

  - Desarrollo backend utilizando PHP nativo.
  - Gestión de información mediante MySQL.
  - Conexión a base de datos utilizando PDO.
  - Uso de consultas preparadas para mayor seguridad.
  - Manejo de sesiones y control de usuarios.
  - Organización modular de archivos PHP.
  - Separación de estilos CSS y scripts JavaScript.
  - Implementación de componentes interactivos mediante JavaScript.
  - Diseño responsive compatible con diferentes dispositivos.

  ## Seguridad y Buenas Prácticas

  - Prevención de inyección SQL mediante consultas preparadas.
  - Validación de datos ingresados por usuarios.
  - Manejo seguro de conexiones mediante PDO.
  - Separación de responsabilidades entre lógica, estilos y scripts.
  - Código organizado para facilitar futuras mejoras.
